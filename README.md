# Times of India – Daily News Digest

A fully automated **Daily News Digest Generator** that fetches the latest top headlines from **Times of India**, summarizes them using **Google Gemini 2.5 Flash**, and produces a clean, ready-to-send HTML/Markdown digest.

This project was built for Kaggle AI Hackathon / Agentic AI workflows and demonstrates:
- RSS feed ingestion  
- LLM-based multi-headline summarization  
- Agent-style pipeline orchestration  
- Clean and production-ready notebook structure  

## Features

- ✔ Fetch live news from **Times of India RSS feeds**  
- ✔ Clean & structure headlines  
- ✔ Summarize using **Google Gemini 2.5 Flash**  
- ✔ Multiple fallback and safety-handling mechanisms  

##  Tech Stack

| Component | Technology |
|----------|------------|
| Language | Python |
| LLM | Google Gemini 2.5 Flash |
